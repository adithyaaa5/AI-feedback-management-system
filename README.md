# AI Feedback Management System

An AI-powered Feedback Management System built using Flask, MySQL, HTML, CSS, JavaScript, and Chart.js.

## Features

- User feedback submission form
- Admin login dashboard
- Search feedback records
- Delete feedback entries
- AI sentiment analysis
- Sentiment visualization using charts
- Modern responsive UI

## Tech Stack

- Python (Flask)
- MySQL
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run

### Clone repository

git clone https://github.com/your-username/ai-feedback-management-system.git

### Install dependencies

pip install flask mysql-connector-python

### Create database

Create MySQL database:

ai_feedback_db

Then create table:

feedback

### Run project

python app.py

Open:

http://127.0.0.1:5000
